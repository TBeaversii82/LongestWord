Book Stats - Star Coder Week 2 Challenge

For your challenge: Select a book and print out the longest word in each book. There are a total of THREE books. For extra credit print out the total word count and the most used word
Note: The book text is contained as constants in the attached js file.
Things you need to do
  • Wire up the buttons to call a function. The dropdown has 2 buttons. Add another one.
  • Create a function to find the longest word. Create a way to process all three books through one function
  • Return the results of the function to the screen.
